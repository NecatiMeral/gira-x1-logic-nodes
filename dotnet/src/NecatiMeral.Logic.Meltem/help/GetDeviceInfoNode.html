Beschreibung

Dieser Baustein ruft alle Daten eines Meltem-Lüftungsgerätes gebündelt ab und stellt diese zur Verfügung. Die Abfrage erfolgt über die Modbus-Schnittstelle (derzeit nur RTU-over-TCP).

Eingänge

Um Daten abzurufen, muss der Trigger-Eingang geschaltet werden. Weitere Eingänge sind:

Bezeichnung Porttyp Beschreibung

Trigger

Bool

Triggert das abrufen der Daten

IP-Adresse

Text

Die IP-Adresse des abzufragenden Meltem-Gerätes

Port

Text

Der Port des abzufragenden Meltem-Gerätes

Geräte-ID

Integer

Die Modbus Slave-ID des abzufragenden Meltem-Gerätes

Ausgänge

Bezeichnung Porttyp Beschreibung

Fehlermeldung

Boolean

0 = Gerät OK; 1 = Fehler

Frostschutzfunktion

Boolean

0 = nicht aktiv; 1 = aktiv

Luftfilterwechsel-Anzeige

Boolean

  • 0 = Luftfilterwechsel-Zeit nicht abgelaufen
  • 1 = Luftfilterwechsel-Zeit abgelaufen

Zeit bis Luftfilterwechsel

Integer

Die verbleibende Zeit bis zum Luftfilterwechsel in Tagen.
