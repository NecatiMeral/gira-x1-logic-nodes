Beschreibung

Dieser Baustein dient zur Umsetzung von Schlüssel-Wert-Paaren abhängig von einem Eingang (Schlüssel oder Wert).

Eingänge

Bezeichnung Porttyp Beschreibung

Eingang

Text

Gibt den Auszuwertenden Schlüssel an.

Tausche Schlüssel/Wert

Boolean

Aktiviert oder Deaktiviert das vertauschen der Schlüssel mit den dazugehörigen Werten.

Trennzeichen

Text

Gibt das Trennzeichen der Schlüssel-Wert-Paare an.

Ausgänge

Bezeichnung Porttyp Beschreibung

Ausgang

Text

Gibt den Wert (oder Schlüssel, wenn `Reverse` aktiviert ist) aus.

Beispiele

In diesem Beispiel werden die textuellen Werte aus der KWB-Comfort-Online Plattform in numerische Werte für eine Speicherung der Daten in einer Influx-DB gewandelt.

Für die Übersicht wurden hier erst die numerischen Werte angegeben und der Parameter Tausche Schlüssel aktiviert. So werden die ankommenden Zeichenfolgen in numerische Werte gewandelt.
