Beschreibung

Mit diesem Baustein kann man mithilfe des ADS-Protokolls (Automation Device Specification) der Firma Beckhoff Daten in ein TwinCAT system schreiben. Dieses System wird unter anderem in Wohnraumlüftungen der Firma Brötje verwendet.

Eingänge

Um Daten zu schreiben, muss der Eingang-Eingang geschaltet werden. Weitere Eingänge sind:

Bezeichnung Porttyp Beschreibung

Eingang

Any

Schreibt die angegebenen Daten entsprechend der Konfiguration in das TwinCAT system. Der Typ des hier bereitgestellten Werts muss mit dem Datentyp-Eingang übereinstimmen.

Remote-Adresse

Text

Die Adresse des SOAP-API-Hosts.

NET ID

Text

Hiermit ist die AMS NET ID, oder auch die TwinCAT system ID gemeint.

Dies kann zum Beispiel wie folgt aufgebaut sein: `192.168.178.24.1.1`

Port

Integer

Portnummer. Die TwinCAT Geräte-ID.

Index-Gruppe

16-Bit Integer

Index Group der Daten, die geschrieben werden sollen.

Index-Offset

16-Bit Integer

Index Offset der Daten, die geschrieben werden sollen.

Datentyp

Auswahl

Der zu sendende Datentyp.

Bezeichnung Porttyp Beschreibung

Text

Im Fehlerfall: Detailinformationen zur Diagnose.
